Welcome to LAPA-NG's documentation!
================================

LAPA-NG (Language Pattern Analyser - Next Generation) is a digital tool for analyzing patterns in spelled language sounds in historical Dutch theatre plays. It converts digitized early modern Dutch theatre plays into (presumed) phonetic script (SAMPA) using a codified ruleset for transliteration.
