API Reference

This page provides detailed documentation for the LAPA-NG Python API, a system for phonetic transcription of Dutch text using rule-based pattern matching.

Core Components

Rules and Matchers

The core of LAPA-NG is its rule-based matching system that converts written text into phonetic sounds.

Core matching functionality for phonetic transcription.

class lapa_ng.rules.matchers.Matcher(Protocol)

Protocol defining the interface for rule matchers.

match(word: str, start: int) → MatchResult | None

Match a substring of a word against a specific rule.

Parameters:

  • word (str) – The word to match against

  • start (int) – Starting position in the word

Returns:

MatchResult if successful, None otherwise

class lapa_ng.rules.matchers.MatchResult

Result of a successful rule match.

Parameters:

  • word (str) – Original word being matched

  • start (int) – Starting position of the match

  • matched (str) – The substring that was matched

  • phonemes (str) – Phonetic transcription for the matched substring

  • remainder (str) – Remaining part of the word after the match

Text Cleaning

Text can be cleaned prior to phonetic transcription using a configurable pipeline.

Text cleaning pipeline functionality.

lapa_ng.pipeline.clean.create_pipeline(*functions: callable) → callable

Create a text cleaning pipeline from a sequence of cleaning functions.

Parameters:

functions – One or more cleaning functions to apply in sequence

Returns:

Combined cleaning function

Available Cleaners:

lapa_ng.pipeline.clean.ensure_text(text: str | None) → str[source]

Ensure that the text is a string and not None. Returns an empty string if input is None.

lapa_ng.pipeline.clean.strip_spaces(text: str) → str[source]

Strip whitespace from the beginning and end of the text.

lapa_ng.pipeline.clean.to_lowercase(text: str) → str[source]

Convert the text to lowercase.

lapa_ng.pipeline.clean.strip_accents(text: str) → str[source]

Remove diacritical marks (accents) from characters in a string.

Example Usage:

from lapa_ng.pipeline.clean import (
    create_pipeline,
    ensure_text,
    strip_spaces,
    to_lowercase,
    strip_accents
)

# Create a pipeline that:
# 1. Ensures text is not None
# 2. Strips whitespace
# 3. Converts to lowercase
# 4. Removes accents
cleaner = create_pipeline(
    ensure_text,
    strip_spaces,
    to_lowercase,
    strip_accents
)

# Use the pipeline
cleaned_text = cleaner("  Café au Lait  ")
# Result: "cafe au lait"
