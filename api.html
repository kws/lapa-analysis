API Reference

This page provides detailed documentation for the LAPA-NG Python API, a system for phonetic transcription of Dutch text using rule-based pattern matching.

Core Components

Rules and Matchers

The core of LAPA-NG is its rule-based matching system that converts written text into phonetic sounds.

Core matching functionality for phonetic transcription.

lapa_ng.rules.matchers.translate(word: str, part_matcher: Matcher) → Generator[ContextualMatchResult, None, None]

Translate a word into phonemes using the given matcher.

This function attempts to match the entire word against the rules and yields a ContextualMatchResult for each match found. If no rule matches a character, it yields a ‘silent’ match with empty phonemes.

Parameters:

  • word (str) – The word to translate

  • part_matcher (Matcher) – The matcher to use for rule matching

Returns:

Generator yielding ContextualMatchResult for each match or non-match

Regex Rules

Rules can be defined using regular expressions for flexible pattern matching, with support for character classes and YAML-based rule specifications.

Regular expression based rule specifications.

class lapa_ng.rules_regex.rules.RegexRuleSpec

Specification for a regular expression based rule.

Parameters:

  • id (str) – Unique identifier for the rule

  • pattern (re.Pattern) – Compiled regular expression pattern

  • replacement (str) – Phonetic replacement string

  • meta (dict[str, Any]) – Additional metadata about the rule

class lapa_ng.rules_regex.rules.RegexMatcher(Matcher)

A matcher that uses regular expressions for pattern matching.

Parameters:

  • id (str) – Unique identifier for the matcher

  • rule (str) – Regular expression pattern

  • replacement (str) – Phonetic replacement string

  • meta (dict[str, Any]) – Optional metadata about the rule

The matcher supports the following character classes: - [:vowel:] - Matches any vowel (aeiouy) - [:consonant:] - Matches any consonant (bcdfghjklmnpqrstvwxz) - [:digit:] - Matches any digit (0123456789) - [:punctuation:] - Matches punctuation (.,!?:;)

Rules starting with ^ are treated as prefix rules and only match at the start of words. Rules must contain a capturing group (pattern) to specify which part of the match to replace.

lapa_ng.rules_regex.rules.load_matchers(rule_file: str | Path) → tuple[RegexMatcher, ...]

Load regex matchers from a YAML file containing rule specifications.

Parameters:

rule_file – Path to YAML file containing rule specifications

Returns:

Tuple of RegexMatcher objects

Example YAML format:

- id: "rule1"
  pattern: "^([:vowel:]+)"
  replacement: "V"
  meta:
    description: "Match initial vowels"

NAF Processing

Text can be read from NAF (NLP Annotation Format) XML files using a streaming parser.

NAF file processing functionality.

class lapa_ng.naf.WordForm[source]

Represents a word form element from a NAF file.

Parameters:

  • text (str) – The text content of the word form

  • attributes (dict[str, str]) – Dictionary of XML attributes associated with the word form

lapa_ng.naf.parse_naf(naf_file: str) → Generator[WordForm, None, None][source]

Parse a NAF file and yield WordForm objects.

This function uses a streaming XML parser to efficiently process large NAF files. It yields WordForm objects for each word form element found in the text section.

Parameters:

naf_file – Path to the NAF file to parse

Returns:

Generator yielding WordForm objects

Example Usage:

from lapa_ng.naf import parse_naf

# Process a NAF file efficiently
for word_form in parse_naf("example.naf"):
    print(f"Text: {word_form.text}")
    print(f"Attributes: {word_form.attributes}")

Text Cleaning

Text can be cleaned prior to phonetic transcription using a configurable pipeline.

Text cleaning pipeline functionality.

functions – One or more cleaning functions to apply in sequence

Returns:

Combined cleaning function

Available Cleaners:

lapa_ng.pipeline.clean.ensure_text(text: str | None) → str[source]

Ensure that the text is a string and not None. Returns an empty string if input is None.

lapa_ng.pipeline.clean.strip_spaces(text: str) → str[source]

Strip whitespace from the beginning and end of the text.

lapa_ng.pipeline.clean.to_lowercase(text: str) → str[source]

Convert the text to lowercase.

lapa_ng.pipeline.clean.strip_accents(text: str) → str[source]

Remove diacritical marks (accents) from characters in a string.

Example Usage:

from lapa_ng.pipeline.clean import (
    create_pipeline,
    ensure_text,
    strip_spaces,
    to_lowercase,
    strip_accents
)

# Create a pipeline that:
# 1. Ensures text is not None
# 2. Strips whitespace
# 3. Converts to lowercase
# 4. Removes accents
cleaner = create_pipeline(
    ensure_text,
    strip_spaces,
    to_lowercase,
    strip_accents
)

# Use the pipeline
cleaned_text = cleaner("  Café au Lait  ")
# Result: "cafe au lait"
